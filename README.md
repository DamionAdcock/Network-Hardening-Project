While I was unable to finish the setup due to time constraints, this is a detailed walkthrough of my steps taken including what didn't work and why. I was able to complete the project with another router and wireless access point, however that was a less streamlined setup and since the router has less RAM and storage it only worked for a few days at a time before requiring reset.
